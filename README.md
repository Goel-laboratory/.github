# 🧬 Goel Laboratory

## 🔬 About the lab

The Goel Laboratory, based at the **Peter MacCallum Cancer Centre** in Melbourne,
Australia 🇦🇺, was founded by Associate Professor Shom Goel in 2019. We work at the
intersection of cell cycle biology, epigenetics, and immunology in cancer 🧫, with the
goal of turning these discoveries into new therapies — particularly for breast cancer 🎗️.

🌐 Learn more at https://www.goellaboratory.com/our-research

## 📂 What you'll find here

This organisation hosts the code and analysis pipelines accompanying our papers.
The aim is reproducibility and transparency ✨: where possible, each publication has
its own repository containing the scripts, notebooks, and analysis code used to
generate the figures and results reported in the manuscript.

📖 Each repository's own `README` documents its contents, dependencies, and how to
reproduce the analyses.
